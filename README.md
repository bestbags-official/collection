## ✨ المميزات الرئيسية (Features)

* **🎭 تجربة تسوق ثلاثية الأبعاد:** واجهة تفاعلية تعتمد على تأثيرات العمق والزجاج لجذب الأنظار.

* **📩 تكامل الواتساب الذكي:** نظام طلب فوري يرسل تفاصيل المنتج (الاسم، السعر، والوصف) للبائع تلقائياً.

* **💻 تصميم متجاوب (Responsive Design):** توافق تام مع جميع الشاشات من الهواتف إلى الشاشات الكبيرة.
